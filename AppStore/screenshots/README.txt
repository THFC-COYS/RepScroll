App Store Screenshots
=====================

Generate placeholder frames:
  python3 scripts/generate_screenshot_frames.py

Before App Store submit, replace placeholders with real iPhone captures:

Required sizes (iPhone):
  • 6.7" (1290×2796) — iPhone 15 Pro Max / 16 Pro Max
  • 6.5" (1284×2778) — iPhone 14 Plus (optional)
  • 6.1" (1179×2556) — iPhone 15 Pro (optional)

Recommended screens:
  1. Home — streak ring + protected apps
  2. Challenge — active camera session with rep count
  3. Gate — blocked app unlock flow
  4. History — stats + weekly chart
  5. Paywall — subscription plans

Tip: Use Xcode → Open Developer Tool → Simulator, or capture on device
with Status Bar clean (Settings → Developer → Status Bar → Clean).
